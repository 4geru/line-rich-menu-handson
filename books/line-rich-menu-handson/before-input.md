# リッチメニューってなに？

LINE 公式 bot の下に出てくるタッチできる部分です。
LINE以外の公式botを調査したところ、60%以上(38/50)のbotでリッチメニューを利用しています。
リッチメニューを使うことで、Bot機能だけでなく、アプリのような表現が可能になります。

## リッチメニューの利用例
リッチメニューを活用しているbotはたくさんあります。
小さいサイズ・大きいサイズと色々あります。
