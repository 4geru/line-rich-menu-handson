# リッチメニューってなに？

LINE 公式 bot の下に出てくるタッチできる部分です。
公式botを調査したところ、60%以上(38/50)のbotでリッチメニューを利用しています。
リッチメニューを使うことで、アプリのような表現が可能になりました。
